Download weights with

```
wget https://pjreddie.com/media/files/yolov3.weights -P yolo/
```

Usage 

```
usage: yolo_opencv.py [-h] -i IMAGE [-c CONFIG] [-w WEIGHTS] [-cl CLASSES]
                      [-t CONFIDENCE_THRESHOLD] [-nsm CONFIDENCE_NSM]
yolo_opencv.py: error: the following arguments are required: -i/--image
```
